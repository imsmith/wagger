Dashboard Layout

The home screen shows system health across all apps and providers at a glance. Which layout communicates status fastest?

A

Grid Matrix

Apps as rows, providers as columns. Each cell is a colored status indicator (green/amber/red). Like a build status board. Entire state visible in one dense grid.

Dashboard — Grid Matrix

Click any cell to jump to that app's config view for that provider. Click app name for detail.

B

App Cards with Inline Badges

Each app is a card with route count, tags, and a row of provider badges. Cards are scannable but take more vertical space. Better for fewer apps with more detail per app.

Only providers with snapshots shown. Click badge to jump to config view. Click app name for detail.

C

Hybrid — Summary Bar + App List
